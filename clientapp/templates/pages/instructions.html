Login

Login is handled with the Persona Observer API.

The app lets Persona manage user state between page refreshes and displays a wait message while synchronization happens. Because routing should otherwise be handled by the client application, this screen should really only be seen by a user when they first visit the app.

To see this screen in action, refresh any page. To force a delay add ?personaLagMs=3000 or some other millisecond value to the url.
