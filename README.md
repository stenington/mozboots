moonboots sandbox
=================

## Quick Start

```bash
  npm install
  npm start
```

Then navigate to http://localhost:3001 and explore from there.

Alternatively, this app can run on [Heroku](https://devcenter.heroku.com/articles/getting-started-with-nodejs#deploy-your-application-to-heroku).
